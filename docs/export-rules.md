# Export Rules

## Export Goal
Allow reviewed finance data to be shared outside the app.

## Export Columns
At minimum export:
- sourceCode
- label
- currentValue
- priorValue
- varianceValue
- variancePercent
- referenceTag
- commentaryText

## Export Modes
- all rows
- rows with commentary only
- rows with reference tags only

## Important
Exports must preserve source code and row ordering.
