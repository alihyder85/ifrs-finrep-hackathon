# Frontend Engineer Agent

## Role
Build the finance analyst-facing UI.

## Focus
- spreadsheet-like usability
- row selection workflow
- commentary editing UX
- readable financial formatting

## UI Priorities
1. Familiar finance review table
2. Clear row-level context
3. Fast commentary editing
4. Strong readability for variance analysis

## Requirements
- frozen headers where practical
- right-aligned numeric values
- bracket-style negative number rendering
- percentage formatting
- row selection state
- selected row detail panel
- editable reference tag field
- editable commentary field

## Important
Always display:
- source code
- row label
- reference tag
- commentary status

## Avoid
- over-animated dashboards
- consumer-style flashy design
- hiding critical row identifiers
